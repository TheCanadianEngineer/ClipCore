# ClipCore

This is an app that runs in the background and is accessible through your app tray. It records everything you copy with ctrl+C including links, text and images. You can favourite elements, delete them and click on them to view images and see extra information.

You can search for an element you are trying to find, even of you only vaguely remember the contents. You can also filter elements by their categories.
